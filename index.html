    FAQ: Perguntas frequentes sobre criação de site

    Um site é um conjunto de páginas web acessíveis através de um único endereço da web, conhecido como URL, ou seja que compartilham o mesmo domínio. Os sites podem ter diversas finalidades, desde servir como a presença online de indivíduos, empresas ou organizações, até possibilitar a venda de produtos ou prestação de serviços. As páginas web que compõem um site podem conter vários elementos, como texto, imagens, vídeos, animações, entre outros recursos interativos.

    Um sistema ou um software é um conjunto de instruções que devem ser seguidas e executadas por um mecanismo, seja ele um computador ou um celular, É o termo usado para descrever programas, aplicativos para celulares e etc...criando códigos de que chamamos de linguagem de programação para instruir o que uma máquina deve fazer.
